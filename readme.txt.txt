Количество бросков - количество бросков воображаемой кости, на которой записаны
номера звуков русского языка(номера каждый раз разные)
На вопрос "consonants, vowels or all?' при запуске программы возможно три ответа:
'consonants' - только согласные
'vowels' - только гласные
'all' - все звуки
характеристика звука вводится следующим образом:
губно-губной;срединный аппрокcимант;звонкий;твёрдый - для согласных
передний ряд;верхне-средний подъём - для гласных
программа также показывает мат. ожидание и дисперсию броска кости, это просто по приколу сделано
чтобы закончить игру, введите "стоп" вместо характеристики звука - 
программа выведет на экран ваш счёт
Удачи на экзамене!
